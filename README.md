# hn-clone
A clone of Hacker News using Ruby/Rails

## Development

#### Requirements:
  Ruby/Rails
  Postgres

1. Clone this repo
2. `cd` into the project directory
3. `bundle install`
4. Make sure you have a running instance of Postgres
4. `rake db:setup`
5. `rails s`
to get the development server running
